# wicoder
auto generador de código básico

agregar el repositorio

<repository>
			<id>jitpack.io</id>
			<url>https://jitpack.io</url>
</repository>
    
agregar la dependencia

<dependency>
			<groupId>com.github.basewicoder</groupId>
			<artifactId>wicoder</artifactId>
			<version>319dd434a3</version>
</dependency>
